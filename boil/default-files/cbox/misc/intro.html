Welcome to adminApp!  Home button above will take you back here.
 
adminApp contains both a web-browser and a database spread-sheet panel.
 
You can interact with these panels to automate most of your website and database.
Start Using adminApp by entering your evenflow web sites url above in the address bar.
Log into quickorder/ folder.
Add some item to your shop.
When viewing or editing a category of items on your website, you will see the AppDB link on your left.
You will only see these AppDB links if you are using this adminApp browser.
It will toggle adminApp to Database/Spread-Sheet panel with all database records from that category.
When your done editing the spreadsheet, just click the Upload Records buttonc
Cnfirm your changes by going back to Web Browser panel and clicking
left menu Refresh llink in the product category page..
The quickorder website uses the Custom Title you give to the sites database in your  Database Query Box  database list.
You can use the Add Database button in the DB Query Box to add your sites database.
Dont forget to use the Reload Database Information button after adding database; to test if database is connected properly.
This Custom Title chosen must be entered into your Shop Settings web page for the websites left menu AppDB links to work.
