this is the project build folder and contains most of the scripts and files needed to build project.
The defaut-files folder contains copies of project files that are added or edited when using adminApp.
On build, the actual adminApp files are deleted and are replaced with copies of these default-files.

The main build script is boil.bat. lets you select what you want to build.

The tools folder contains a fork of the yahoo yuicompressor to compress the website javascript and css files 
that are found in the webdroid/assets/quickorder web-android folder. 
See the tools/yuicompressor4 README for more info.
tools/packaging folder contains configuration files for packaging apps.

The android build folder is at webdroid/bin. See webdroid/README
