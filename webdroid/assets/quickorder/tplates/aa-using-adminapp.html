Using adminApp

The Database/Spread-Sheet panel does not have this sites database on the database list.
You can toggle this browser panel with  the Databse/Spread-Sheet panel using the top left corner button.
This website uses the Custom Title you give to this sites database in your  Database Query Box list.
If this database is already set up on the  Database Query Box list,its Custom Title must be saved to your shop settings page. in order for the left menu AppDB links to work.
Or add this sites database by clicking the Add Database button in the Database Query Box.
NOTE:
To add this sites database to this adminApp you must use the database name, database username and db password that is set up in  file _p/do.php
The Database Custom Titles are unique and are used in cases where the same database name is used on different servers, i.e. local, remote.
The Web Folder Url is the folder url to the _p/ folder ex: _p/
The Custom Title must be saved to your shop settings page.
