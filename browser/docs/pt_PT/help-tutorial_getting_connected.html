Tutorial - Fazer Ligação
   
 
 Este tutorial está assumindo que você já tiver instalado adminApp e fez upload da pasta qcomms para seu website.
 
  Para verificar se adminApp está se conectando corretamente ao arquivo qcomms.php, você deve adicionar um dos seus bases de dados do seu site para a lista de Base de Dados adminApp.
  Abra a caixa [Adicionar Base de Dados] ou  usando o menu Ferramentas na janela principal ou no menu Base de Dados na caixa de pesquisa.
Também pode usar o botão [Adicionar Base de Dados] na caixa de pesquisa.
 
 Isso só acrescenta a informação do base de dados para a lista de Base de Dados na caixa de pesquisa.
 Isso não adiciona um base de dados para o seu site nem alterar o seu base de dados.
 
  Introduza as informações necessárias na Caixa de Adicionar Base dados (veja aqui) e clique em Guardar.
  Será pedido para actualizar as informações de adminApp do base de dados (estrutura de tabela, nomes de domínio, etc)
  Se adminApp conecta ao arquivo qcomms.php e obtém suas informações de base de dados com sucesso, você verá uma lista de todas as tabelas do seu base de dados na caixa de pesquisa.
 
  Escolha uma das tabelas que contêm pelo menos um registro e clique no botão Executar SQL. Deve exibir os últimos 50 registros dessa tabela na janela principal de folha de calculo/Spread-Sheet.

  Agora pode editar os registros na folha de calculo
