 Quantos resultados (numero de registos/rows no seu Base de Dados) posso pedir de uma véz?
  Se está a trabalhar em um servidor local, pode tentar até 5000, dependendo do número de campos na tabela, o tamanho total dos dados.
Se está a fazer ligação a um  Web site remoto, tenha em consideração o tempo que seria necessário para mostrar os 5000 resultados em uma página web e dividir por 2 ou 3.
Embora adminApp comprime os dados, tente começar com 50 ou 100 resultados, e veja os resultados.
 
 
 Porque estou a ter problemas com a ligação á pasta qcomms no meu website?
  Por favor, veja o erro que é mostrado no pop up de alerta.
Você também pode abrir um browser e ir para a página web qcomms.php,
(Onde você carregou o qcomms pasta para o seu site),
e veja se você está recebendo algum erro php.
Ex: http://www.mywebsite.wow/qcomms_folder_in_here/qcomms/qcomms.php
