Caixa de Pesquisa (base de dados)
   
Esta caixa de pesquisa é onde você vai, para pesquisar seus bases de dados
Os resultados são exibidos na janela de folha de calculo principal.

O ultimo base de dados utilizado será automaticamente selecionada na lista.
A caixa de pesquisa, também se lembra de seu tamanho e localização no ecrã.
Pode redimensioná-lo na horizontal, mais não na vertical.
 
Abaixo está uma listagem e descrição de todos os componentes nesta caixa de pesquisa.
 
 
 
 
 
Base de Dados Lista:      Contém uma lista de bancos de dados do Web site que foram adicionados com a janel Adicionar/Editar.
Por padrão, o DemoDatabase é adicionado e não pode ser editado ou eliminado.
     
Configurações BaseDados Botão: Abre a janela Adicionar/Editar base de dados, permitindo que você edite as configurações do base de dados seleccionado.
     
Adicionar Base de Dados Botão:   Abre a janela Adicionar/Editar base de dados, permitindo que você  adicione um dos seus bases
 de dados do seu web-site para a lista.
     
Actualizar Info BaseDados Botão:     Atualiza as listas de tabelas e de campos. Por exemplo, No caso de você alterar seu base de dados, ou seja, adicionar ou excluir uma tabela ou campo. (tem que estar ligado, faz ligação ao ficheiro qcomms).
     
Informação Rótulo:   O texto mostrado acima das tabelas e listas de campos. Mostra a base de dados atual e o nome da tabela.
     
Tabelas BaseDados Lista Drop-down:   Lista de todas as tabelas no base de dados selecionado. Você pode usar a lista para selecionar uma das tabelas ou você pode clicar em uma das tabelas na lista.
     
Campos BaseDados Lista Drop-down :   Exibe todos os campos da tabela selecionada. São duplicadas para facilitar a busca na tabela.
     
Numero de Resultados Campo Texto:   O número de resultados de pesquisa para mostrar. Tome em consideração que, dependendo de sua conexão com a web, mostrando um grande número de resultados, assim como mais de 500, pode demorar algum tempo. Compare como em exibir 500 resultados em um editor web MySQL como o phpMyAdmin
     
Comando SQL Query Campo Texto:   A área de texto que exibe o comando SQL para o seu base de dados.
Este texto SQL é atualizado e auto-completa quando você alterar ou selecionar uma nova tabela ou um nome de campo de tabela, ou alterar o seu critério de busca na entrada de campos da tabela.
Você também pode inserir seu próprio texto SQL para esta área de texto (o que não se destina a uma consulta SQL com mais de 500 linhas)
     
Executar SQL Botão:    Conecta-se a sua pasta qcomms (upload para website) e executa o comando SQL da área de texto.
     
Esconder apos executar... Checkbox:    Caso marcada, esconde esta caixa de pesquisa uma vez que os resultados são retornados e exibidos na janela principal (folha de calculo).
