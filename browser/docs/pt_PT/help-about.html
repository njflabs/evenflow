Sobre adminApp
   
 
adminApp é um aplicativo escrito em Java que lhe dá suporte de folhas de calculo (spread-sheets),
com ligação a base de dados MySQL e não requer drivers.
Na maioria dos casos, a funcionalidade obtida através de uma página web não é suficiente para editar e manipular os seus web-sites normalmente manipulados por base de dados MySQL.
Se estiver familiarizado com folhas de cálculo ou phpMyAdmin, utilizando adminApp não deve ser tão difícil.
