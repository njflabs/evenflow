Guia Rapida
   
 
  adminApp  é um programa java  que faz ligaçao aos ficheiros na pasta qcomms (tem que fazer upload da pasta qcomms para seu website - veja Instalar ).
 
  Certifique-se que adminApp está a fazer ligação com sucesso para a pasta qcomms no seu website.
Confirme isto em tentar adicionar uma das suas bases de dados do seu website à lista de BaseDados adminApp (ver Fazer Ligação).
 
  Se adminApp faz ligação á pasta qcomms e amostar a informação de seu base de dados com sucesso, verá uma lista de todas as tabelas na Caixa de Pesquisa
 
  A Caixa de Pesquisa é utilizada para pesquisar as mesas/tabelas do seu BaseDados.
Todos resultados são exebidos na  folha de calculo (veja Pesquisar Registos).
 
  Depois pode editar os registos na Folha de Calculo. Pode utilizar as opções de Cortar, Copiar, Preencher e outras.
Quando terminar de editar os registos, podem ser uploaded ao seu BaseDados ou no total ou individualmente (veja Editar e Uploading registos).
