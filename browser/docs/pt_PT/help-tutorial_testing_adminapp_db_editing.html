Tutorial - Testar adminApp
   
 
 Este tutorial está assumindo que já está ligado ao Base de Dados do seu website.
 
  O texto sql em baixo vai eliminar, drop if exists, se existir, e crie um TestTable no BaseDados atualmente selecionado em sua lista BaseDados.
Nota que o primeiro campo test_table (testFieldA), é auto incrementado (1, 2, 3, 4, etc.)
Copie (arraste e selecione texto e utilize as teclas Ctrl+ C), o texto abaixo de cor vermelha para a área de texto [Comando SQL (Query)] localizado na caixa de pesquisa,
(A área de texto que exibe a instrução SQL  para o seu base de dados), e clique no botão [Executar SQL].
   
  DROP TABLE IF EXISTS test_table;
CREATE TABLE test_table (
testFieldA int(11) auto_increment,
testFieldB varchar(50),
testFieldC varchar(50),
testFieldD int(12),
UNIQUE KEY testFieldA (testFieldA)
) ENGINE=MyISAM DEFAULT CHARSET=latin1 COLLATE=latin1_general_ci;

INSERT INTO test_table VALUES ('', 'yes', 'no', 5);
INSERT INTO test_table VALUES ('', 'yes', 'no', 5);
   
   
   
  Isto irá criar uma tabela, "test_table",  em seu database.
Para confirmar isso, clique em botao [Atualizar BaseDados] também encontrado na caixa de pesquisa (lado superior direito).
Agora você deve ver o test_table na lista de tabelas encontrada na caixa de pesquisa (lado esquerdo).
Clique em test_table, ou selecione-o na lista e clique no botão [Executar SQL].
Uma vez que este test_table contém apenas dois registros, eles serão exibidos em sua folha de calculo.
Para adicionar mais alguns registros:
Botão direito do mouse sobre a folha de calculo e selecione [Adicionar linhas] e no menu pop-up. digite "10".
Isto irá adicionar 10 linhas em branco para esta folha de calculo.
   
  Continuado em baixo.....
 
 
 
  Clique na célula na coluna testFieldB, número da linha 2, o valor é a palavra "sim", e arraste para baixo até que você selecione todas as células em branco nesta coluna.
Depois clique sobre elas com o botão direito do rato e selecione a opçao [Preencher] que aparece no pop-up.
Isto irá preencher todas as células em branco com a palavra "sim". Faça o mesmo para a coluna testFieldC. Na coluna testFieldD, alterar o valor de "5" para "10". Em seguida, faça a mesma coisa, selecione as células vazias, clique direito sobre as células, e no pop-up e selecione a opção [Preencher].
   
 
A coluna testFieldA não está a ser preenchido, pois é um campo auto-incrementado e introduz o proximo número automaticamente, i.e., 3, 4, 5, ...
E como estamos fazer adição de registros, não há necessidade de editar.
   
 
  Uma vez que o preenchimento é feito nas células como mostrado acima, clique no botão [Upload Registo(s)] encontrado no canto superior direito . Isto irá adicionar estes registros nas celulas para o test_table em seu base de dados.
   
  Uma caixa de alerta, (pop up), deve abrir e mostrar o número de registros enviados.
Pode confirmar se os registros foram enviados através de clicando no [test_table] na lista de Tabelas na Caixa de Pesquisa.
Em seguida, clicar no botão[Executar Query].
Os novos resultados devem aparecer na planilha.
