Caixa de Editar Registo
   
Você pode abrir essa caixa de edição de registro quando:
Você seleciona o [Editar Registo] no top-menu [Dados].
Ou pelo botão direito do mouse sobre a folha de calculo e selecione [Editar Registo]
Além disso, um duplo clique sobre o número da linha (coluna cinza no lado esquerdo), vai abrir esta caixa de Editar Registro.
Você pode editar os valores de campo de registro, selecionando o campo para editar e inserir o novo valor no campo de edição.
Uma vez que você terminar a edição, você pode fazer upload o registro para o base de dados ou adicioná-la à lista de Comandos SQL Pendentes.
Veja descrição de todas as opções abaixo.
Clicar duas vezes sobre o valor do campo
e uma ampla área de texto será aberto.

Todos os valores de campo em branco irá mostrar o valor "isnull".
Não mudar se deseja que o valor fique nulo.
 
Menu Registo
Registo-> [Editar Campo]:   Alterna para a área de texto maior que lhe permite editar o valor do campo selecionado. Clicando duas vezes sobre o valor do campo também abre esta area de texto maior
Registo-> [Faça Upload de Registo..]: Faz Upload deste registo para seu base de dados.
 Leia como os registros são enviados usando o "Replace into tablename" comando SQL.
Registo-> [Adicionar a Lista de SQL Pendente]: Adiciona este registro para a lista de Comandos SQL Pendentes encontrado no fundo da janela folha de calculo
Registo-> [Fechar]: Fechar esta janela.
Componentes
Esta caixa de editar registro cria duas listas paralelas (uma dos Nomes dos campos e outra com os valores destes campos) para o registro específico. Observe que o registro exibido na linha selecionada da folha de calculo não é atualizado quando você terminar de editar ou fazer upload deste registro. Você deve fazer isso manualmente nesta versão. Ou apenas refazer a solicitação de pesquisa para atualizar seus registros de folha de cálculo.
