Janela Adicionar/Editar Base de Dados
   
 
A janela de  Adicionar / Editar base de dados basicamente, acrescenta um dos base de dados de seu web-site para o lista de base de dados na caixa de pesquisa.
Pode adicionar todos os base de dados do seu Web site a esta lista.
Deve fornecer a URL da pasta qcomms (ver instalação), e informações do usuário do base de dados.

Abaixo está uma listagem e descrição de todos os componentes nesta Janela de Adicionar/Editar base de dados.
   Nota: isto só irá funcionar se você tiver feito upload da pasta qcomms para o seu web site (veja aqui). Não é necessário fazer upload deste ficheiro em cada nova versao de adminApp. Veja o menu [Ferramentas -> Actualizar ficheiro qcomms] em baixo.
 
   Isto não adiciona um base de dados para o seu web site. Isso só acrescenta a informação do base de dados do seu site para a lista de base de dados da caixa de pesquisa.
 
 
 
 
Ao adicionar um Base de Dados para a lista de base de dados na caixa de pesquisa , você deve fornecer os seguintes dados:
Ferramentas-> [Actualizar ficheiro qcomms]:      Não é necessário fazer upload deste ficheiro em cada nova versao de adminApp. Basta usar este menu para fazer atualização(experimental).
     
Titulo:      Título personalizado para este base de dados do site. (Números de letras e espaços somente). Ex: Meu remoto DB
     
URL Pasta Web:   O endereço do site da pasta onde a pasta qcomms está localizada.
Ex: http://www.blablamysite.com/dbstuff/
(Neste caso, a pasta qcomms está localizada na pasta dbstuff).
Não inclui a pasta qcomms na url, apenas a pasta em que reside e termine com a barra /.
     
     
Nome de Base de Dados:   Nome de Base de Dados.
     
Database Host:   Database host. Geralmente, localhost.
     
Database Username:   Username de Base De Dados.
     
Database Password:   Password de Base de Dados.
     
     
    Login de pasta protegida por senha 
(pasta qcomms protegido por senha com ficheiros htaccess/.htpasswd).
Optional - but highly recommended.
Username:   Username da pasta protegida.
     
Password:   Password da pasta protegida.
