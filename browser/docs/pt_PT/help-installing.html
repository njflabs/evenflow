Instalar adminApp
   
 
Depois de ter instalado adminApp no seu computador:
Abra a pasta de instalação adminApp e localize a pasta website_uploads.
Está geralmente em: c: \ Program Files \ adminApp \ website_uploads.
Também pode abrir esta pasta através de ambiente de trabalho,  Iniciar -> Todos os Programas - adminApp -> website_uploads
Faça upload para seu website a pasta qcomms encontrada dentro da pasta website_uploads.
Ex: http://www.mysite.com/qcomms/
 adminApp faz ligação aos ficheiros dentro da pasta qcomms (no seu website),  para interagir com os seus Base De Dado(s).
 Você só tem que fazer upload da pasta qcomms uma vez, e não para cada Base de Dados no seu servidor web.
 Chmodd tanto qcomms e qcomms subpasta / dbqs para  777 
 Embora adminApp faz encription basica da sua ligação, você pode querer proteger sua pasta qcomms com senha (md5).
 Veja utilizar password protected pastas.
 
Agora você pode testar a configuração e tentar uma ligação.
Veja adicionando um BaseDados do seu website para a Lista BaseDados de adminApp.
