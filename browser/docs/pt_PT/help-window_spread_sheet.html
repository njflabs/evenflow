Janela Principal - Folha de Calculo
Esta é a janela principal que se abre quando você inicia adminApp.
Para amostrar resultados nesta folha de calculo, primeiro tem que procurar por resultados do seu base de dados através da Caixa de Pesquisa. Uma vez exibidas, você pode editá-los com algumas das capacidades de folha de calculo abaixo mecionados.
Mais funções serão adicionados em versões futuras.		  
Veja descrição de todas as opções abaixo.
 
 
Lista de Menus Topo
Ficheiro -> [Abrir]:   Abre ficheiro .cvs selecionado (experimental).
Ficheiro -> [Guardar]:   Guarda ficheiro .cvs selecionado (experimental).
Ficheiro -> [Guardar Como]:   Guarda Como ficheiro .cvs selecionado (experimental).
Ficheiro -> [Fechar]:   Fecha o programa.
Editar -> [Copiar]: Copiea o texto selecionado na folha de calculo.
Editar-> [Colar]: Cola para a folha de calculo texto copiado.
Editar -> [Preencher Coluna]: Preenche as células selecionadas em uma coluna com o valor da primeira célula selecionada.
Ferramentas-> [Pesquisar Base de Dados]: Abre a caixa de pesquisa, permitindo a procura no seu base de dados para que os resultados sejam exibidos nesta folha de calculo.
Ferramentas->  [Adicionar Base De Dados]: Opens the  box, allowing you to add one of your website databases to the Search Box database list.
Abre a caixa de Adicionar/Editar base de dados, permitindo que você adicione uma das bases de dados do seu site á lista na caixa de pesquisa.
Dados -> [Editar Registo]: Abre a caixa de Editar Registro, que lhe permite editar o registro seleccionado.
Dados -> [Faça Upload Registros Seleccionados]: Faz upload do registro selecionado para o base de dados
 Leia como os registros são enviados usando o comando MySQL "Replace into tablename".
Dados ->  [Faça Upload de todos Registros]: Faz upload do registros na folha de calculo para o base de dados
 Leia como os registros são enviados usando o comando MySQL "Replace into tablename".
Dados -> [Adicionar Fila]: Adiciona uma linha de células no fundo desta folha de calculo.
Dados -> [Adicionar Fila(s)]: Adiciona linha(s) de células no fundo desta folha de calculo.
Dados -> [Eliminar Fila(s)]: Elimina as filas selecionadas da folha de calculo.
 Isto não elimina os registros do seu base de dados, somente desta folha de calculo.
 
Folha de Calculo
Informação Rótulo:   O texto mostrado acima da folha de calculo. Mostra o seu base de dados atual e o nome da tabela
     
Pesquisar Base de Dados  Botão: Abre a caixa de pesquisa de base de dados, permitindo que você procure em seu base de dados para que os resultados sejam exibidos nesta folha de calculo.
Faça Upload Registros(s) Botão:   Faz upload de todos os registros na folha de calculo para seu base de dados.
     
Barra de Progresso:   Enconta se no canto direito superior. As barras de progresso, (encontrado principalmente no lado superior direito da maioria das janelas) irá piscar enquanto adminApp ésta a fazer uma ligação para o seu site. Por enquanto, eles não vão lhe mostrar um progresso, percentualmente, porque a maioria das conexões DSL (retornar 500 registros), levará menos de 4 segundos. A característica porcentagem pode ser adicionada no futuro.
     
Folha de Calculo:   Este é o local onde os registros do base de dados são exibidos depois de procurar seu base de dados através da caixa de pesquisa.
Depois de editar os resultados nesta folha de calculo, pode fazer upload de todos os resultados para o seu base de dados ou pode fazer upload de cada registro individualmente.
Os registros também podem ser editados individualmente e guardado na lista de comandos SQL pendentes (upload mais tarde).
Para editar registros individualmente, pode pelo botão direito do mouse sobre a folha de calculo e selecione [Editar Registro]
Além disso, um duplo clique sobre o número da linha (coluna cinza no lado esquerdo), vai abrir a caixa de Editar Registro.
Tambem veja Editar Registros.
     
Comandos SQL Pendentes Lista:   A lista de Comandos SQL Pendentes é encontrado no fundo desta folha de calculos. Basicamente, em vez de fazer upload de cada registro selecionado após a edição, você pode enviar o registro para esta lista.  Uma vez que terminar de editar os registros selecionados e adicioná-los a esta lista, então você pode fazer upload de todos de uma vez.
