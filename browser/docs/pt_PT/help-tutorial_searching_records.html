Tutorial - Pesquisar Registros
   
 
Este tutorial está assumindo que você já está conectado ao seu base de dados do seu site
 
  Todas Pesquisas aos base de dados, incluindo Executar SQL, é feita através da Caixa de Pesquisa
  Pode introduzir texto SQL no campo de texto [Comando SQL (Query)]. Tambem actualiza-se automaticamante conforme a criteria de pesquisa é alterada.
 
A instrução de consulta padrão na área de texto SQL é mostrar as últimas 50 registros da tabela de base de dados selecionado
 i.e.; select * from DemoTableD order by DemoFieldD1 desc limit 50;
 
Os resultados então aparecerá na folha de calaculo onde você pode editar e fazer upload desses registos / resultados.
