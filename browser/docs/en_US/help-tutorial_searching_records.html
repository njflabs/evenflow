Tutorial - Searching Records
   
 
 This tutorial is assuming you area already connected to your website database.
 
  All database searching, including running MySQL queries is done through the Database Query/Search Box.
  The query text in the search box will take any sql query, or auto complete one as you select your search criteria.
 The default query statement in the SQL text area is to show the last 50 records of the selected database table.
 i.e.; select * from DemoTableD order by DemoFieldD1 desc limit 50;
The results will then show up on your spread sheet where you can edit and upload these records/results.
