Add/Edit Database Form
   
 
The Add/Edit Database window basically adds one of your website databases to the SearchBox list.
You can add all your website databases to the list.
You must supply the web folder url of the qcomms folder (see Installing), and database user information.
 
Below is a listing with a description of all the components.
 
   Note, this will only work if you have uploaded the qcomms/ folder to your website (see here). It is not necessary to upload this file for every new version of adminApp. See the Tools menu below.
 
   This does not add a database to your website.  This only adds the website database info to the Search Box database list.
 
 
 
 
In adding each database name to the search box database list, you must supply the following data:
Tools -> [Update qcomms file]:      Automatically uploads and updates your qcomms file (beta).
     
Custom Title:      Custom Title for this website database. (Letters numbers and spaces only). Ex: My Remote DB
     
Web Folder Url:   The website folder address where the qcomms folder is located.
Ex: http://www.blablamysite.com/dbstuff/
(In this case the qcomms folder would be in the dbstuff/ folder).
Do not include the qcomms in the Url, only the folder in which it resides and ended with forward slash /.
     
     
Database Name:   Your database name.
     
Database Host:   Your database host. Usually localhost.
     
Database Username:   Your database username.
     
Database Password:   Your database password.
     
     
    Password protected folder login
(password protected qcomms folder with .htaccess/.htpasswd files).
Optional - but highly recommended.
Username:   the protected web folder username.
     
Password:   the protected web folder password.
