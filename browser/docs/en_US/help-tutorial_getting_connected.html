Tutorial - Getting Connected
   
 
 This tutorial is assuming you have already installed adminApp and uploaded the qcomms folder to your website.
 
  In order to check if adminApp is connecting correctly to the uploaded qcomms.php file, you must add one of your website databases to your database list.
  Open up the Add Database Box either buy using the Tools Menu in the main window or the Database Menu in the Search Box.
You can also use the Add Database button in the Search Box.
 
 This only adds the database info to the Search Box database list.
 This does not add a database to your website nor alter your database.
 
  Enter the required informtion in the Add Database Box (see here), and click Save.
  You will be prompted to reload the database information (table structure, field names, etc).
  If adminApp connects to the qcomms.php file and fetches your database information  successfully, you will see a listing of all the tables in your database in the Search Box
 
  Select one of the tables that contain at least one record and click the Run Query  button. It should display the last 50 records from that table in the Spread-Sheet window.
 
  Now you can edit the records in the spread-sheet window.
