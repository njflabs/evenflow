Tutorial - Editing Records
   
 
This tutorial is assuming you already connected to your website database, have searched your database for records, and they are displayed on your spread-sheet.
 
  There are multiple ways to edit the database records on the spread-sheet.
  You can edit them directly on the spread-sheet or you can edit them individually by right clicking over the spread-sheet row and selecting Edit Record. Also, double clicking on the row number ( grey column on left hand side), will open up the Record Edit box.
 
  Editing and deleting rows on the spread-sheet will not alter your database.
Your database will only be altered if you upload the records.
Deleting rows on a spread sheet comes in handy if say you searched for 50 records and modified only like 25.
You would then delete the rows you haven't modified and upload the rest.
  There are also a few options if you want to add a record to your table.

Select the table that you would like to add a record to.
In the Number of Results, put 3 and hit Run Query.
You should now see the last 3 records from that table in the spread-sheet.
In the spread-sheet top menu,  select [Data] -> [Add Row].
It will add a blank row to the bottom of your spread sheet.
Fill in the record fields on the spread sheet and click upload records.

Keep in mind that adminApp assumes that your database tables have the first field in your tables as a unique index field, auto increment.
So if you were to delete the auto-increment number (set it to blank) on the spread-sheet, when you upload the records, the blank field is usually treated as a new record with MySql.

 If your table has no records, by default it will set a demo record by copying the field names as values to the spread-sheet.
 This record is not in your database. It is just to facilitate adding a record to the table.
