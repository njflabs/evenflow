Database Query/Search Box
   
   
 
The Database Search/Query box is where you go to for searching your databases.
The results are then displayed in the main spread-sheet window.
You can open this search box bu clicking on the [Search Database] button found on top right hand corner of main spread sheet screen.
You can also use the top [Tools] menu.
 
The last database used will be automatically selected in the database list.
The search-box also remembers its size and location on screen.
You can resize it horizontally but not vertically.
 
Below is a listing with a description of all the components.
 
 
 
Database List:      Contains a list of website databases that were added with the Add/Edit database box.
By default, the DemoDatabase is added. You can not edit, search or delete this DemoDatabase.
     
Database Settings Button: Opens the Add/Edit database box, allowing you to edit the database settings.
     
Add Database Button:   Opens the Add/Edit database box, allowing you to add one of your website databases to the list.
     
Reload Database Information Button:   Refreshes the table and field lists. In case you alter your database, i.e.: add/delete a table or field to/from table.
     
Information Label:   Text shown above the tables and fields lists. Displays your current debase and table name.
     
Database Tables Drop-down List:   List all tables in the currently selected database. You can either use the drop-down list to select one of your database tables for searching or you can click on one of the tables in the list below it.
     
Database Fields Drop-down List:   Displays all the fields in the currently selected table. They are displayed a few times to facilitate the searching within the table.
     
Number of Results Text Input:   The number of search results to show. Take into consideration that depending on your web connection, displaying a large number of results, like over 500, may take some time. Compare to displaying 500 results on a web based MySQL editor like phpMyAdmin.
     
SQL Query Text Area:   The text area that displays the actual SQL statement for your database.
This SQL text is updated and auto-completes when you change either select a new table or a table field name or alter your search criteria in the table fields input.
You can also enter your own SQL text into this text area (this is not intended for a SQL query with more than 500 lines).
     
Run Query Button:    Connects to your qcomms folder and executes the SQL text query in the SQL Query  text area.
     
Hide on Query Checkbox:    If checked, hides this search box once results are returned and displayed in the spread-sheet window.
You can always re-open this search box either by right clicking over spread sheet and selecting Search Database or click the Search Database button in the spread-sheet window.
