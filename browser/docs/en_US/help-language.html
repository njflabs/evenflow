adminApp in your Language
   
 
By default, adminApp and documentation come in the english language.
Currently the only other language in beta is portuguese.
Changing the adminApp application to your language:
In the top [Tools -> Language] menu of adminApp you can select one of the available languages.
 
 You can only have one instance of adminApp open, and you must close adminApp and re-launch for changes to take place.
 All help files will also try to default to the selected language when clicking on any Help menu within adminApp.
 If the selected language documentation is not found ex: c:\adminApp\docs\pt_PT it will default to the en_US english language.
 
To go directly to your selected language help documentation in your computers Start menu; Start -> All Programs -> adminApp -> Help, right click over it, select Properties, and in the Target field, change it from the main html file to your language folders file, see above
 
If you would like to help out in refining/translating the documentation or adminApp language strings, let us know.
You can find the adminApp language strings file in: the cbox\props folder. The APBundle_[locale].props files can be edited in a text editor.
You may want to make a back-up before editing. You can test the strings by relaunching adminApp after making changes.
