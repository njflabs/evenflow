Installing adminApp
   
 
Once you have installed adminApp on your computer:
Open up the adminApp installation folder and find the website_uploads folder.
This is usually in: c:\Program Files\adminApp\website_uploads.
You can also open this folder through your computers  Start -> All Programs -> adminApp -> website uploads folder shortcut
Upload the qcomms folder found inside website_uploads to your web site.
Ex: http://www.mysite.com/qcomms/
 adminApp connects to the files in the qcomms folder (on your website),  to interact with your mysql database(s).
 You only have to upload the qcomms folder once, and not for every database on your web server.
 chmodd both qcomms and the subfolder qcomms/dbqs to  777 
 although adminApp encrypts connection data, you may want to password protect your qcomms folder (md5).
 See using password protected folders.
 
Now you can test the setup and try your connection by adding one of your website databases to the adminApp Database List .
