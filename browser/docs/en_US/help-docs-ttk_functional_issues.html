  Functional Issues
   
 
adminApp uses tab delimitation for displaying your database results on the spread sheet. 
The "Replace into tablename" MySQL command is used for uploading records into database.
It is assumed that the first field in you database table is a unique index field.
 Always use a test table for fully testing adminApp before fully using adminApp on your database.
The connection progress bars,  (mainly found on the top right hand side of most windows) will flash while adminApp is making a connection to your website. For now, they will not show you a progress, percentage wise, because most connections (DSL returning 500 records), will take less than 4 seconds. The percentage feature may be added in the future.
 Always use a test table for fully testing adminApp before fully using adminApp on your database.
