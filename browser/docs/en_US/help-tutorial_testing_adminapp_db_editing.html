Tutorial - Testing
   
 
 This tutorial is assuming you area already connected to your website database.
 
  The below sql text will, drop if exists, and then create a testTable in the currently selected database on your database list.
Note that the first field in test_table (testFieldA), is auto incremented (1, 2, 3, 4 etc).
Copy (drag and select text and use Ctrl+ C keys), the below red colored text  to the SQL Query text area located in the search box,
(The text area that displays the actual SQL statement for your database),  and click the [Run Query] button.
   
  DROP TABLE IF EXISTS test_table;
CREATE TABLE test_table (
testFieldA int(11) auto_increment,
testFieldB varchar(50),
testFieldC varchar(50),
testFieldD int(12),
UNIQUE KEY testFieldA (testFieldA)
) ENGINE=MyISAM DEFAULT CHARSET=latin1 COLLATE=latin1_general_ci;

INSERT INTO test_table VALUES ('', 'yes', 'no', 5);
INSERT INTO test_table VALUES ('', 'yes', 'no', 5);
   
   
  This will create a test_table in your website database.
To confirm this, click the [Reload Database Information] button also found in search box (top right hand side).
You should now see the test_table in the tables list found in search box (left hand side).
Click on test_table, or select it from the list, and click the [Run Query] button.
Since this test_table only contains 2 records, they will be displayed on your spread-sheet.
  Lets add a few more records.
Right click over the spread-sheet and select [Add Rows] from the pop-up menu.
Enter "10" in the text input area. This will add 10 blank rows to your spread-sheet.
  Continue below.....
 
 
 
  Click on the cell in the testFieldB column, row number 2, value is the word "yes",
and drag down until you select all the blank cells in this column.
Right-click over them and select the [Fill] option in the pop-up.
This will fill all blank cells with the word "yes".
Do the same for the testFieldC column.
In the testFieldD column, change the value from "5" to "10".
Then do the same thing, select empty cells, right click for pop up and select the [Fill] option.
   
 
 The testFieldA column is not being filled because it is an auto-incremented field in your database.
 It will number itself, 3, 4, 5, etc.
 And since we are adding records, there is no need to edit.
   
 
  Once your done filling in the cells as shown above, click the [Upload Record(s)] button found on top right hand corner of spread-sheet.
This will add the extra records to the test_table in your web-site database.
   
  An alert box should pop open showing you the number of records uploaded.
You can confirm if the records have been uploaded by selecting or clicking on the test_table in the SQL/Search Box.
Then clicking the [Run Query] button.
The new results should show up on the spread-sheet.
