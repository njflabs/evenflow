Main Spread-Sheet Window
   
This is the main window that opens up when you launch adminApp.
Following will open up the SQL Search Box.
In order to display results in this spread-sheet, you will first have to search for database results through the search box.
Once results are displayed, you can edit them with some of the spread sheet capabilities listed below.
More functions will be added in future versions.
 
Below is a listing with a description of all the components.
 
 
 
 
Menu List
File -> [Open]:   Opens selected cvs file (experimental).
File -> [Save]:   Saves selected cvs file  (experimental).
File -> [Save As]:   Saves As selected cvs file  (experimental).
File -> [Quit]:   Closes the adminApp Program.
Edit -> [Copy]: Copies the selected text/cells on the spread-sheet.
Edit -> [Paste]: Pastes the copied text/cells to the spread-sheet.
Edit -> [Fill Column]: Fills the selected cells in a column with the first selected cell value.
Tools-> [Search Database]: Opens the database search box, allowing you to search your database for results to be displayed on spread-sheet.
Tools -> [Add Database]: Opens the Add/Edit database box, allowing you to add one of your website databases to the Search Box database list.
Data -> [Edit Record]: Opens the Edit Record box, allowing you to edit the selected row record.
Data -> [Upload Selected Record to Database]: Uploads the selected record row to your database.
 Read how records are uploaded using the "Replace into tablename" MySQL command.
Data ->  [Upload All Record to Database]: Uploads all records on the spread-sheet to your database.
 Read how records are uploaded using the "Replace into tablename" MySQL command.
Data -> [Add Row]: Adds a cell row to the bottom of the spread-sheet.
Data -> [Add Row(s)]: Adds cell row(s) to the bottom of the spread-sheet.
Data -> [Delete Row(s)]: Deletes the selected rows from the spread-sheet.
 This does not delete the records from your database, only from the spread-sheet.
 
 
Spread Sheet
Information Label:   Text shown above the spread-sheet. Displays your current database and table name.
     
Search Database Button: Opens the database search box, allowing you to search your database for results to be displayed on spread-sheet.
Upload Record(s) Button:   Uploads all records on spread-sheet to your database.
 Read how records are uploaded using the "update" MySQL command.
     
Progress Bar:   Upper right hand corner.
The connection progress bars,  (mainly found on the top right hand side of most windows) will flash while adminApp is making a connection to your website. For now, they will not show you a progress, percentage wise, because most connections (DSL returning 500 records), will take less than 4 seconds. The percentage feature may be added in the future.
     
Spread-Sheet:   This is where the database records are displayed after searching your database through the database search box.
After editing the results on the spread-sheet, you can upload all the results on the spread-sheet to the database or you can upload each record individually.
Records can also be edited individually and saved to the Pending Queries list (upload later).
To edit records individually, you can open up the Edit Record box either by double-clicking on the row number in the grey left hand column (showing the row numbers), right clicking over the row record and selecting Edit Record, or using the Menu -> Data -> Edit Record. Also see Editing Records.
     
Pending Queries List:   The Pending Queries list is found on the bottom of the spread-sheet.
Basically instead of uploading each selected record after editing it, you can send the record row to this list.
For now sending the record row to his list can only be done through the Edit Record box.
Once you are done editing your selected records and adding them to this list, you can then upload them all at once.
