Security Issues
   
 
adminApp makes connections to your website (qcomms folder), to get and upload database records.
Although adminApp uses a basic algorithm to encrypt the data,
we strongly suggest that you password protect the qcomms/ folder or the parent folder.
In other words, try not to put the qcomms folder in your website root directory.
Ex: http://www.mydomian.wow/dbstuff/qcomms/
 
SSL or proxy support have not been added yet to adminApp.
