Quick Start
   
 
  adminApp is a java spread sheet program that connects to the qcomms php files (you must upload the qcomms folder to your website - see Installing).
 
  Make sure that adminApp is connecting successfully to your websites qcomms/ folder.
Confirm by adding one of your website databases to the adminApp database list (see Getting Connected).
 
  If adminApp connects to the qcomms  files and fetches your database information successfully, you will see a listing of all the tables in your database in the Search Box
 
  The Search Box is used to search your database tables.
All database results are displayed on the spread-sheet (see Searching Records).
 
  You can then edit the records on the spread sheet, using fill, copy, paste and other options.
Once you are done editing the records, they can be uploaded individually or in total (see Editing and Uploading records).
