Record Edit Box
   
 
You can open up this record edit box when:
You select the [Edit Record] option either from the spread-sheet top Data menu.
Or by right-clicking over the selected row in the spread-sheet and select the [Edit Record] option from the right-click popup.
Double clicking over the grey cell on the spread sheets right hand side will also open up this Record Edit box.
You can edit the record field values by selecting the field to edit and entering the new value in the Field Edit text input.
If the field value is a long text string, you can double click over the field value and a wider text area will open up.
Once you are done editing, you can upload the record to your database or add it to the Pending Uploads list.
Below is a listing with a description of all the components.
 
Double clicking on a field value
will open up this larger text area.

 All blank field values will show the value "isnull".
 If you want to upload a blank value, do not change, it will set isnull to blank automatically.
 
Record Menu
Record -> [Edit Field]:   Will switch to the larger text area allowing you to edit the selected field value.
Double clicking on the field value will also switch to the larger text area.
Record -> [Upload Record to Database]: Uploads the record to your website database.
 Read how records are uploaded using the "Replace into tablename" MySQL command.
Record -> [Add to Pending Uploads]: Adds this record to the Pending Queries list found on the bottom of your spread-sheet.
Record -> [Close]: Closes this Record Edit box.
 
Components
The Record Edit box creates 2 parallel lists of the field names and field values for the specific database table record.
Note that the selected row record displayed on the spread sheet is not updated when you finish editing/uploading this record.
You must do that manually in this version. Or just redo the search request to refresh your spread-sheet records.
